# 🚀 SWC C++ Bindings

> *Because sometimes we need to wrap Rust tools in C++ which we'll later call from JavaScript... programmers, right?* 🤦‍♂️

TypeScript transpiler, compiler, and minifier for C++ — brought to you by someone who thought "you know what this Rust-based tool needs? More languages!"

These bindings expose SWC's lightning-fast TypeScript/JavaScript pipeline directly to C and C++, without installing Node.js on your beautifully organized machine (that probably has 47 versions of Node anyway).

> [!important]
> This project is like my attempts at baking: experimental and not recommended if you want predictable results.

## ✨ Features

- ⚡ **Absurdly Fast:** Transpiles TypeScript faster than you can say "why am I wrapping a Rust tool in C++?"
- 🔗 **C/C++ Compatible:** Clean C API with a generated `swc.h` header, wrapped in a `swc::` namespace
- 🧩 **Zero Runtime:** No Node.js or TypeScript toolchain needed (one less dependency to break your build!)
- 🗜️ **Minification:** Compress and mangle JS with SWC's production-grade minifier
- 🦀 **Powered by Rust:** Yes, we're using a Rust-based tool, wrapping it in C++, to process JavaScript... software engineering at its finest!

## 🔧 API

The header `swc.h` is auto-generated by `cbindgen` on every build. All functions live in the `swc` namespace.

### Transpile

```c
// Transpile TypeScript/TSX from a string — returns JS or NULL on failure
char* swc::transpile(const char* filename, const char* code);

// Transpile from a file path (deprecated — use compile_file)
char* swc::transpile_file(const char* filepath);
```

### Compile

```c
// Compile a TypeScript/JS file to JavaScript
char* swc::compile_file(const char* filepath, char** error);

// Compile TypeScript/JS from a string
char* swc::compile_js(const char* code, char** error);
```

### Minify

```c
// Minify a JavaScript file (compress + mangle)
char* swc::minify_js_file(const char* filepath, char** error);

// Minify JavaScript from a string
char* swc::minify_js(const char* code, char** error);
```

### Memory Management

```c
// The most important function that everyone forgets to call
void swc::free_string(char* s);

// For the const-correct among us
void swc::free_const_string(const char* s);
```

## 📝 Example

```cpp
#include "swc.h"
#include <iostream>

int main(int argc, char const *argv[]) {
    const char* ts_code = R"(
        interface User { name: string }
        const greet = (user: User) => `Hello ${user.name}!`;
    )";

    // Transpile a string
    char* js = swc::transpile("example.ts", ts_code);
    if (js) {
        std::cout << js << std::endl;
        swc::free_string(js);
    }

    // Compile a file and minify the result
    char* error = nullptr;
    char* compiled = swc::compile_file(argv[1], &error);

    if (compiled) {
        char* minified = swc::minify_js(compiled, &error);
        std::cout << minified << std::endl;
        swc::free_string(minified);
        swc::free_string(compiled);
    } else if (error) {
        std::cerr << "Error: " << error << std::endl;
        swc::free_string(error);
    }

    return 0;
}
```
